# Reglas globales del proyecto

- Este repositorio es la fuente de verdad.
- No reescribir la arquitectura sin justificación.
- No considerar terminado un trabajo si:
  - no compila
  - fallan tests
  - falta actualizar docs/current-release.md
- Toda tarea debe pertenecer a un release del roadmap.
- Antes de implementar, leer:
  - docs/ai-context.md
  - docs/roadmap.md
  - docs/current-release.md
- Si hay ambigüedad, volver al Analista.
- Si falla build o test, corregir y repetir.
