# AI Context

## Objetivo del sistema
El objetivo general del sistema GestAI Comercio es ofrecer una plataforma SaaS multi-tenant para gestionar de punta a punta la operación comercial de una empresa, combinando la capa de administración de tenant/usuarios con los procesos operativos del negocio (maestros, ventas, compras, stock, facturación y reportes) en una misma solución web.

## Stack
- Backend: .NET 9
- Frontend: Blazor
- DB: SQL Server
- Arquitectura: Clean Architecture + CQRS + MediatR
